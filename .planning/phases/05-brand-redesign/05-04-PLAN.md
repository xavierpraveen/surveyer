RECHARTS NOTE: Recharts SVG props (fill, stroke, colors array) cannot use CSS variables — they must be concrete hex values. Use hex values from spec Section 2 palette:
- Brand:          #6366F1
- Brand muted:    #EEF2FF
- Accent:         #8B5CF6
- Success:        #10B981
- Warning:        #F59E0B
- Error:          #EF4444
- Fg-muted:       #64748B

SCORE THRESHOLD COLORING (for DimensionBarChart bars and DepartmentHeatmap cells):
- score >= 75: success fill (#10B981)
- score >= 50: warning fill (#F59E0B)
- score < 50:  error fill (#EF4444)

KPI METRICS:
- Card: bg-surface border border-border rounded-lg shadow-sm p-5
- Metric number: text-2xl font-extrabold tracking-snug text-fg tabular-nums
- Metric label: text-xs text-fg-subtle font-medium uppercase tracking-[0.07em]
- Metric change indicator: text-xs tabular-nums text-success-text (positive) or text-error-text (negative)

FILTER BAR:
- Label: text-sm font-semibold text-fg mb-1
- Select/input: spec form input class (border border-border rounded-md bg-surface px-3 py-2 text-sm text-fg focus:outline-none focus:ring-2 focus:ring-indigo-200 focus:border-border-focus)
- Apply button: Primary variant
- Reset/Clear button: Ghost variant

THRESHOLD PLACEHOLDER:
- Wrapper: bg-brand-muted border border-brand-muted rounded-lg p-5 text-center
- Icon or lock symbol: text-brand text-2xl (or w-8 h-8)
- Message: text-sm text-brand-text font-medium
- Sub-message: text-xs text-fg-muted mt-1

QUALITATIVE THEME PANEL:
- Section heading: text-base font-bold tracking-tight text-fg mb-4
- Theme tag: bg-brand-muted text-brand-text text-xs font-semibold rounded-full px-3 py-1
- Theme count badge: bg-surface-2 text-fg-muted text-xs tabular-nums font-semibold rounded-full px-2 py-0.5 ml-1

TREND LINE CHART:
- Recharts Line stroke: '#6366F1' (brand) for primary series, '#8B5CF6' (accent) for secondary
- Recharts CartesianGrid strokeDasharray="3 3" stroke="#E2E8F0" (border color)
- Recharts XAxis/YAxis tick fill: '#64748B' (fg-muted)
- Wrapper card: bg-surface border border-border rounded-lg shadow-sm p-5

CYCLE SELECTOR (results/CycleSelector.tsx):
- Select element: spec form input class
- Label: text-sm font-semibold text-fg mb-1
- Wrapper: flex items-center gap-3
